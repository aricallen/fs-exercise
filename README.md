# FsExercise

## Setup

```sh
# install dependencies
yarn
```

**for typescript**

```sh
yarn dev # run tsc in watch mode
# new tab
yarn serve:ts # express server
# open http://localhost:3000
```

**for javascript**

```sh
# for javascript
yarn serve:js
# open http://localhost:3000
```
